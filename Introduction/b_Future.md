# That's all Folks!

## Lexy Specifications

The lexy language describes itself with a collection of lexy scenarios and example functions. They define the expected output behavior and even expected compilation and validation errors. These specifications ensure that all lexy compiler implementations in all different languages have the same expected behavior. Have a look at the '[README](https://github.com/lexy-language/lexy-language/blob/main/Specifications/Readme.md)' in the 'Specifications' folder for more information about the specifications.

### Last note

We hope you enjoyed the introduction!

Again, the main goal of Lexy is to encourage governments to publish their Tax Laws in a readable, transparent and executable form!
